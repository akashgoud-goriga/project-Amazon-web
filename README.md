# Amazon Clone – Frontend UI

A responsive front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's web interface, focusing on clean layout design, responsive structure, and modern styling techniques. It was developed as a practice project to strengthen front-end web development skills without using JavaScript or backend technologies.

## Features

* Responsive navigation bar
* Amazon-inspired homepage layout
* Search bar and menu sections
* Hero banner section
* Product category cards
* Footer with multiple sections
* Clean and organized HTML & CSS code

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Purpose

The primary goal of this project is to practice front-end development by recreating a popular e-commerce website interface. It demonstrates an understanding of semantic HTML, CSS styling, layout design, and responsive web development.
